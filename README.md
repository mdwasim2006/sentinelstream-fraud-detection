# SentinelStream – Real-Time Fraud Detection Engine

SentinelStream is a FinTech backend system designed to detect fraudulent financial
transactions in real time using a combination of rule-based logic and machine learning.

## Project Status
Week 1 – Planning & Architecture (Completed)

## Key Features
- Real-time transaction processing
- Rule-based fraud detection
- Machine learning-based anomaly detection
- Redis caching and rate limiting
- Asynchronous background processing

## Technology Stack
FastAPI, PostgreSQL, Redis, Celery, Scikit-learn, Docker, Nginx
